> [!INFO]
> - The goal of this page is to provide a template for the seminar notes. For each [[seminar]] session, a note is created filling in the following information.
> - The note can be created before the seminar session to provide information or materials beforehand, and must be either created or updated after the session (e.g., include youtube recording link).
> - This is a bare template and can be adapted to each session's requirements.

# Materials

This can be one or more of the following:

- attached PDF file: slides, report, paper, screenshots of digial blackboard, etc. Ideally these documents are stored in the website itself, under the current semester's directory. If, for example, you link to your thesis pointing to an external url, please make sure that it remains accessible long-term.
- link to an arXiv paper, a paper from another source (please check that it is freely accessible for non-affiliated people), a blog post of astrobites or similar, etc

# Notes

Here you can write notes about the session such as interesting comments or questions worth giving a thought or a summary of the Q&A/discussion after the presentation.
